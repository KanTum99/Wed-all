การติดตั้ง Termux

วิธีติดตั้ง Termux ให้เสถียรและพร้อมใช้งาน ห้ามโหลดจาก Play Store เพราะเก่าเกินไป

# 1. ดาวน์โหลดจาก F-Droid หรือ Github Releases
pkg update && pkg upgrade
pkg install git python nodejs

แต่ง Termux เท่ๆ

เปลี่ยนธีมเป็น Oh My Zsh ติดตั้งฟอนต์ และจัดการสีสันให้ดูเป็นโปร

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
pkg install neofetch
# แก้ไขไฟล์ .zshrc เพื่อใส่ Theme

สร้างโปรเจค CLI

เริ่มต้นเขียนเครื่องมือ Command Line Interface ด้วย Node.js บนมือถือ

mkdir my-cli-tool && cd my-cli-tool
npm init -y
npm install chalk commander ora

ข้อบกพร่อง & ข้อควรระวัง

อย่างที่กรูบอก... Termux ไม่ใช่ magic box มันมีเรื่องที่ต้องยอมรับ

  • แบตหมดเร็ว: การคอมไพล์โค้ดหรือรันสคริปต์หนักๆ กินแบตมหาศาล
  • Update เจ็บ: บางครั้งอัปเดตแล้วเกิด Bug หรือ Environment พังได้
  • No Root Default: ถ้าอยากเล่นขั้นสูงต้อง Root เครื่องก่อน (Proot ก็ได้แต่จำกัด)
  • Keyboard เล่นงาน: คีย์บอร์ดมือถือไม่เหมาะกับการเขียนโค้ดยาวๆ แนะนำใช้จอยหรือคีย์บอร์ดภายนอก
